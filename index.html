Whaleunit

Docker support for JUnit based integration testing

View on GitHub

Orchestrate your Docker containers

This project will let you manage Docker containers as part of a JUnit intergration test.

Include Whaleunit in your project

Once this project actually exists in a usable form, you can include a dependency in your project. 

<dependency>
  <groupId>cool.arch.whaleunit</groupId>
  <artifactId>whaleunit-impl</artifactId>
  <version>1.0-SNAPSHOT</version>
</dependency>
Whaleunit is maintained by CoolArch
